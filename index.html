Sitevision Public API documentation for Dash

Dash documentation for Sitevision Public API version 2025.01.1.

Installation on MacOS with Dash

Click this link to add the feed into Dash: Dash feed.

Or download the compressed docset, extract it, and add it manually to Dash: Sitevision.docset.tgz.

Don't know what Dash is? See Dash website.

Installation on Windows with Zeal

  1. Start Zeal and open Tools/Docsets.
  2. Click Add feed.
  3. Paste https://hampusn.github.io/sitevision-dash-docs/Sitevision.xml and click OK.

Don't know what Zeal is? See Zeal website.
